# Person API Error Handling

## Описание
Ошибка "Failed to fetch person" на странице `/person/[id]` возникала при проблемах с сетью или TMDB API.

## Решение
Улучшена обработка ошибок в `src/app/api/person/[id]/route.ts`:

1. **Таймауты** - более не логируются как ошибки (ожидаемое поведение при проблемах с сетью)
2. **404** - актер не найден в TMDB возвращает понятную ошибку
3. **Фильмография недоступна** - если не удалось получить credits, возвращаются данные актера без filmography
