# Context: Phase 6 - Доработка страницы статистики /profile/stats

## Decisions

- **UI Pattern**: Использовать тот же стиль плашек, что на странице /profile (блок "Статистика" с плашками Просмотренно, Отложенно, Брошено, Скрыто)
- **Content Types**: 4 типа контента - Фильмы, Сериалы, Аниме, Мульты
- **Layout**: Горизонтальные плашки в верхней части страницы /profile/stats

## Description

В верхней части страницы /profile/stats размещаем 4 плашки с типами контента:
- Фильмы
- Сериалы  
- Аниме
- Мульты

Каждая плашка показывает числовое значение (количество) по данному типу контента.

## Plan 01

В верхней части размещаем 4 плашки с типами контента - Фильмы, Сериалы, Аниме, Мульты (пример берем со страницы /profile, блок Статистика, плашки Просмотренно, Отложенно, Брошено, Скрыто), Выводим цифровые значения по данным типам контента
